# Piscuss

Piscuss is a fast and lightweight open source discussion application.

## Features

- [ ] All Threads
- Channels
- Popular This Week
- Popular All Time
- Solved
- Unsolved
- No Replies Yet
- Leaderboard

## Development

Start project

```shell
python manage.py migrate
python manage.py runserver
```

Create a admin user

```shell
python manage.py createsuperuser
# piscuss / piscuss / piscuss@example.com
```
